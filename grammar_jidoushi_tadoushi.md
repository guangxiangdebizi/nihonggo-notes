# 自动词 vs 他动词详解（じどうし vs たどうし）

> 核心问题：为什么日语要把动词分成「自己动」和「他动」？为什么有的用 **が**、有的用 **を**？

---

## 一、是什么？一句话区分

| | 自动词（じどうし） | 他动词（たどうし） |
|---|-------------------|-------------------|
| 罗马音 | jidoushi | tadoushi |
| 中文 | 不及物动词 | 及物动词 |
| 核心 | **主语自己**发生动作/变化 | **主语对宾语**做动作 |
| 宾语 | 一般**没有**を宾语 | 通常有 **を** 宾语 |
| 例 | ドアが **開く** | ドアを **開ける** |

```
自动词：ドアが 開く。
        Doa ga aku.
        门（自己）开了。

他动词：私が ドアを 開ける。
        Watashi ga doa wo akeru.
        我（把门）打开。
```

---

## 二、为什么日语要分自动词和他动词？

### 2.1 语言在描述「谁负责这个动作」

中文常说「我开门」，一个「开」字，主语、动作、对象全在里面。  
日语要把三件事**拆开说清楚**：

| 中文 | 日语拆法 | 为什么 |
|------|----------|--------|
| 门开了 | ドア**が** 開く | 门是变化主体，没人强调谁干的 |
| 我开门 | 私**が** ドア**を** 開ける | 我是施动者，门是被作用对象 |

**自动词 / 他动词的对立，就是日语在回答：**

> 这个变化，是**事物自己发生**的，还是**有人去作用**它发生的？

### 2.2 和助词系统绑在一起

日语不是只靠语序，而是靠 **が / を / に** 标角色：

```
自动词句：  [变化主体] が  +  自动词
他动词句：  [施动者]   が  +  [对象] を  +  他动词
```

**为什么自动词后面很少接「を」？**  
因为 **を** 标记的是「被直接作用的对象」。自动词描述的是主语**自身的状态变化**，没有「对外施加动作的对象」，所以不用 **を**。

**为什么他动词后面要有「を」？**  
因为他动词语义上就是「对某物做某事」，**を** 正好标这个「某物」。

### 2.3 一对动词，两种世界观

日语里大量 **自他配对**（同一个意思，两个动词）：

| 自动词 | 罗马音 | 他动词 | 罗马音 | 区别 |
|--------|--------|--------|--------|------|
| 開く | aku | 開ける | akeru | 自己开 / 把…打开 |
| 閉まる | shimar | 閉める | shimeru | 自己关 / 把…关上 |
| 始まる | hajimar | 始める | hajimeru | 自己开始 / 使…开始 |
| 止まる | tomar | 止める | tomeru | 自己停 / 使…停 |
| 入る | hair | 入れる | ireru | 自己进 / 把…放进 |
| 出る | der | 出す | dasu | 自己出 / 把…拿出 |
| 付く | tsuk | 付ける | tsukeru | 自己附上 / 把…附上 |
| 消える | kier | 消す | kesu | 自己灭 / 把…灭掉 |
| 壊れる | kowar | 壊す | kowasu | 自己坏 / 把…弄坏 |
| 落ちる | ochir | 落とす | otosu | 自己掉 / 把…弄掉 |

**为什么要成对存在？**  
因为日语需要区分：

- 雨**が** 降る（雨自己下）—— 没人能「を 雨」
- 本**を** 落とす（把书弄掉）—— 必须有人 + を

## 三、怎么判断？—— 三个实用方法

### 3.1 看能不能接「を + 名词」

```
✅ 本を 読む（たどうし）   hon wo yomu
❌ 本を 行く（不成立）

✅ 窓を 開ける（たどうし）  mado wo akeru
✅ 窓が 開く（じどうし）    mado ga aku（主语是窗，不是人）
```

**检验法：** 在动词前加「何を（什么）」问——能答 **を** 宾语的，多半是他动词。

```
何を 読みますか。→ 本を 読みます。  ✅ 他动词
何を 行きますか。→ （不自然）        ❌ 自动词
```

### 3.2 看主语是「人」还是「物/事」

| 主语 | 常见动词类型 | 例 |
|------|-------------|-----|
| 物/事 自己变化 | 自动词 + **が** | 花**が** 咲く / 雨**が** 降る |
| 人 对物做动作 | 他动词 + **を** | 私**が** 花**を** 摘む |

> 注意：自动词主语也可以是人（人**が** 行く），但那是「人自己移动」，不是「对某物做动作」。

### 3.3 看成对动词的词尾规律（常见）

| 规律 | 自动词 | 他动词 | 说明 |
|------|--------|--------|------|
| ～aru / ～eru | 開**く** aku | 開**ける** akeru | 最常见配对 |
| ～mu / ～meru | 止**まる** tomaru | 止**める** tomeru | 五段自 / 一段他 |
| ～su 结尾 | — | 出**す** dasu | 很多他动词以 **す** 结尾 |
| ～eru / ～su | 消**える** kieru | 消**す** kesu | 自 er / 他 su |

**不是 100% 可靠**，但见到 **～aru / ～eru / ～u** 与 **～eru / ～su** 成对时，可以优先猜自他关系。

## 四、为什么助词不同？—— が vs を 的本质

### 4.1 自动词：主语 = 变化本身

```
子供が 走る。
Kodomo ga hashiru.
孩子在跑。（孩子 = 跑的主体）

雨が 降る。
Ame ga furu.
下雨了。（雨 = 下的主体）
```

**が** 标记的是：**谁/什么 在发生这个状态或动作**。

### 4.2 他动词：主语 ≠ 动作对象

```
母が 子供を 叱る。
Haha ga kodomo wo shikaru.
母亲训斥孩子。
  ↑施动    ↑被作用对象

学生が 本を 読む。
Gakusei ga hon wo yomu.
学生读书。
```

- **が** → 施动者（谁在做）
- **を** → 受动者/对象（对谁/对什么做）

### 4.3 同一事件，两种视角

**「窗户开了」—— 两种说法：**

```
窓が 開いた。
Mado ga aita.
窗开了。（强调窗自己开了，不知谁开的）

（私が）窓を 開けた。
(Watashi ga) mado wo aketa.
（我）把窗打开了。（强调是我做的）
```

**为什么日语要分这么细？**  
因为**责任、原因、焦点**不同：

- 自动词 → 描述**现象/结果**
- 他动词 → 描述**人为动作**

## 五、和「花钱」的关系 —— 为什么 使う 是他动词、かかる 是自动词？

（呼应 `grammar_okane_expressions.md`）

| 动词 | 类型 | 句型 | 为什么 |
|------|------|------|--------|
| 使う | **他动词** | お金**を** 使う | 「钱」是被使用的对象 → **を** |
| かかる | **自动词** | お金**が** かかる | 「钱」是消耗物，主语是「旅行/修理」等 → 钱用 **が** 作主语 |
| 払う | **他动词** | 授業料**を** 払う | 「学费」是被支付的对象 → **を** |

```
私が お金を 使った。        ← 人 + を + 他动词
Watashi ga okane wo tsukutta.

旅行に お金が かかった。    ← 事 + に + 钱が + 自动词
Ryokou ni okane ga kakatta.
```

**为什么不能说「お金を かかる」？**  
因为 **かかる** 是自动词，语义是「（时间/金钱）被花费掉」，**钱** 在这个结构里是**主语**，不是 **を** 宾语。

## 六、和「使役态」的关系 —— 为什么自动词使役用を、他动词使役用に？

（呼应 `grammar_causative_shieki_tai.md`）

使役 = 让某人做某事。自他不同，**被使役者的角色**就不同：

### 6.1 他动词使役 → 被使役者 = 执行者 → **に**

```
基本形：子供が 作文を 書く。
        Kodomo ga sakubun wo kaku.

使役形：母は 子供に 作文を 書かせる。
        Haha wa kodomo ni sakubun wo kakaseru.
```

- **書く** 是他动词 → 本来就有「子供が + 作文を」
- 使役后，「子供」仍是**写的人** → 用 **に**（动作接受者/执行者）
- 「作文」仍是**被写的东西** → 仍用 **を**

### 6.2 自动词使役 → 被使役者 = 原主语 → **を**

```
基本形：子供が 公園へ 行く。
        Kodomo ga kouen e iku.

使役形：母は 子供を 公園へ 行かせる。
        Haha wa kodomo wo kouen e ikaseru.
```

- **行く** 是自动词 → 只有「子供が」，没有を宾语
- 使役时，原主语「子供」被「拉进来」当**使役作用对象** → 换成 **を**
- **为什么？** 因为使役者（母）成了新主语，原来的「が主语」降级为「を对象」

### 6.3 还原法（做题用）

```
第一步：使役动词还原成基本形
第二步：看基本形有没有「を + 名词」
        有 → 他动词 → 人 + に
        无 → 自动词 → 人 + を
```

| 使役形 | 基本形 | 类型 | 人 + |
|--------|--------|------|------|
| 書かせる | 書く（作文を） | 他 | **に** |
| 行かせる | 行く | 自 | **を** |
| 読ませる | 読む（本を） | 他 | **に** |
| 泣かせる | 泣く | 自 | **を** |
| 心配させる | 心配する | 自 | **を** |

## 七、和「被动态」的关系 —— 为什么他动词被动保留を？

```
他动词被动：
  本を 読む  →  本が 読まれる
  hon wo yomu  →  hon ga yomareru
  读书       →  书被读

自动词被动：
  雨が 降る  →  雨が 降られる（少见/受限）
  更多用使役被动或其他表达
```

**他动词被动为什么「を」变「が」？**  
原句「本**を** 読む」中，**本** 是动作对象；  
被动句视角转向**本**，它成了句子的主题/主语 → **本が 読まれる**。

**为什么自动词被动少见？**  
自动词本身没有 **を** 宾语，被动化后语义容易别扭（「被走了？」），所以自动词更多走使役、使役被动等别的路径。

## 九、只有他动词 / 只有自动词的情况

### 9.1 常见「只有他动词」

对**外物施加动作**，必须有一个作用对象：

| 动词 | 罗马音 | 例 |
|------|--------|-----|
| 食べる | taberu | ご飯**を** 食べる gohan wo taberu |
| 飲む | nomu | 水**を** 飲む mizu wo nomu |
| 読む | yomu | 本**を** 読む hon wo yomu |
| 書く | kaku | 字**を** 書く ji wo kaku |
| 買う | kau | 本**を** 買う hon wo kau |
| 使う | tsukau | お金**を** 使う okane wo tsukau |

### 9.2 常见「只有自动词」

**自然现象、自发移动、状态变化**：

| 动词 | 罗马音 | 例 |
|------|--------|-----|
| 行く | iku | 学校**に** 行く gakkou ni iku |
| 来る | kuru | 友達**が** 来る tomodachi ga kuru |
| 降る | furu | 雨**が** 降る ame ga furu |
| 咲く | saku | 花**が** 咲く hana ga saku |
| 死ぬ | shinu | 犬**が** 死んだ inu ga shinda |
| 座る | suwaru | 椅子**に** 座る isu ni suwaru |

> **に** 不是 **を**，因为「学校」「椅子」是**目的地/位置**，不是他动词的「作用对象」。

## 十、常见错误与纠正

| 错误 | 原因 | 正确 |
|------|------|------|
| ドア**を** 開く | 把自动词当他动词 | ドア**が** 開く / ドア**を** 開**ける** |
| 雨**を** 降る | 自然现象不能を | 雨**が** 降る |
| 学校**を** 行く | 行く是自动词，学校是目的地 | 学校**に** 行く |
| お金**を** かかる | かかる是自动词 | お金**が** かかる |
| 子供**を** 作文を 書かせる | 他动词使役误用を指人 | 子供**に** 作文を 書かせる |
| 子供**に** 公園へ 行かせる | 自动词使役误用に | 子供**を** 公園へ 行かせる |

## 十一、自测练习

判断自动词（自）还是他动词（他），并填空：

1. 私は 毎日 新聞（　）読む。
2. 電車が 駅に（　）。
3. 母は 子供（　） 野菜（　） 食べさせる。
4. 旅行（　） お金（　） かかった。
5. 窓が 自動的に（　）。（开：開く/開ける）

**参考答案：**

1. **を**（他）— 新聞**を** 読む / Shinbun **wo** yomu  
2. **着く**（自）— 電車が 駅に **着く** / Densha ga eki ni **tsuku**  
3. **に** … **を**（他）— 子供**に** 野菜**を** 食べさせる  
4. **に** … **が**（自）— 旅行**に** お金**が** かかった  
5. **開く**（自，窗自己开）— 窓が 自動的に **開く** / Mado ga jidou-teki ni **aku**  
   （若人打开 → 窓**を** **開ける**）

## 十二、总结卡片

```
┌─────────────────────────────────────────────────────┐
│  自动词（じどうし）                                     │
│    主语自己动 / 状态变化    [物] が  +  自动词           │
│    例：ドアが 開く / 雨が 降る / お金が かかる          │
│                                                     │
│  他动词（たどうし）                                     │
│    主语对对象做动作        [人] が  +  [物] を  +  他动词 │
│    例：ドアを 開ける / 本を 読む / お金を 使う          │
│                                                     │
│  为什么分？ → 区分「自然发生」vs「人为作用」              │
│  为什么助词？ → が=主体/施动  を=被作用对象              │
│                                                     │
│  使役态：他动 → 人に + を  /  自动 → 人を（原が变を）    │
└─────────────────────────────────────────────────────┘
```
