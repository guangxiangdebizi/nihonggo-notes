# 动词使役态详解（どうししえきたい）

> 对应《新标准日语教程》第二册第10课 · 使役助动词「せる・させる」

---

## 一、使役态是什么？

### 1.1 一句话定义

**使役态**（しえきたい）是日语动词的一种变化形式，表示：

> **说话人（或句中主语）让、使、叫、允许某人做某事，或让某事发生。**

中文里常对应「让」「使」「叫」「派」「允许」等，但日语不是另加一个「让」字，而是**直接变动词本身**。

| 基本形 | 使役态 | 中文 |
|--------|--------|------|
| 書く（写） | 書か**せる** | 让（某人）写 |
| 行く（去） | 行か**せる** | 让（某人）去 |
| 食べる（吃） | 食べ**させる** | 让（某人）吃 |
| 来る（来） | 来**させる** | 让（某人）来 |
| する（做） | **させる** | 让（某人）做 |

### 1.2 和中文的对应关系

中文结构：**谁 + 让 + 谁 + 做某事**

```
妈妈让孩子写作业。
  ↑    ↑   ↑
主语  使役 被使役者
```

日语结构：**谁 + 被使役者 + 助词 + 动作（使役形）**

```
母は子供に作文を書かせる。
  ↑    ↑    ↑      ↑
主语 被使役  宾语   使役动词
```

关键差异：日语把「让…做」**压缩进动词变形**，所以必须先掌握变形规则，再掌握助词搭配。

## 二、为什么日语要有使役态？

### 2.1 语言结构上的原因

日语是**黏着语**：语法关系主要靠助词和动词词尾变化来表达，而不是靠语序或独立虚词。

- 被动有「れる・られる」→ 被……
- 使役有「せる・させる」→ 让……
- 可能也有「れる・られる」（同形异义）→ 能……

如果不用使役态，就要用冗长的说法，例如：

```
❌ 母は子供に「作文を書く」ことをさせる（不自然）
✅ 母は子供に作文を書かせる（自然）
```

### 2.2 语义上的两种「让」

使役态其实覆盖两层意思，教材常分别称为：

| 类型 | 含义 | 典型场景 | 例 |
|------|------|----------|-----|
| **使动** | 强制、命令、促使 | 父母管孩子、上司派任务 | 宿題をさせる |
| **使役** | 允许、许可 | 请假、先走一步 | 先に失礼させていただきます |

同一个「書かせる」，可以是「逼孩子写」，也可以是「允许/安排孩子写」——**具体语气看上下文**，不是变形不同。

## 三、使役态怎么变？（接续规则）

### 3.1 核心公式

使役态由助动词 **「せる」「させる」** 接在动词**未然形**后面构成：

| 动词类型 | 接续 | 例 |
|----------|------|-----|
| 五段动词 | 未然形 + **せる** | 読む → 読ま + せる → **読ませる** |
| サ变动词 | 未然形 + **せる** | する → さ + せる → **させる** |
| 一段动词 | 未然形 + **させる** | 食べる → 食べ + させる → **食べさせる** |
| カ变动词 | 未然形 + **させる** | 来る → 来 + させる → **来させる** |

### 3.2 完整变形表

| 分类 | 词例 | 词干 | 未然形 | 使役态 |
|------|------|------|--------|--------|
| カ行 | 書く | か | か**か** | か**かせる** |
| ガ行 | 泳ぐ | およ | およ**が** | およ**がせる** |
| サ行 | 話す | はな | はな**さ** | はな**させる** |
| タ行 | 立つ | た | た**た** | た**たせる** |
| ナ行 | 死ぬ | し | し**な** | し**なせる** |
| バ行 | 飛ぶ | と | と**ば** | と**ばせる** |
| マ行 | 読む | よ | よ**ま** | よ**ませる** |
| ラ行 | 降る | ふ | ふ**ら** | ふ**らせる** |
| ワ行 | 歌う | うた | うた**わ** | うた**わせる** |
| 一段 | 食べる | た | た**べ** | た**べさせる** |
| 一段 | 教える | おし | おし**え** | おし**えさせる** |
| サ变 | する | — | **さ** | **させる** |
| カ变 | 来る | — | **こ** | **来させる** |

### 3.3 为什么五段用「せる」、一段用「させる」？

这是历史语法遗留的**音便与助动词接法**问题，初学阶段不必深究词源，记住实用规律即可：

1. **五段动词未然形较短**（一个音节：読**ま**、書**か**），后面接 **せる** 就能读顺。
2. **一段动词未然形已是完整词干**（食**べ**、見**る→見**），若只加「せる」会变成「食べせる」，发音拗口；因此统一加 **させる**，形成「食べさせる」这种双「せ」结构。
3. **する・来る** 作为不规则动词，分别变为 **させる・来させる**，需要单独记忆。

> **记忆口诀：** 五段 + する → **せる**；一段 + 来る → **させる**。

### 3.4 「せる・させる」本身怎么活用？

它们按**一段动词**活用，所以敬体、过去式、否定都能正常变化：

| 形式 | 読ませる | 食べさせる |
|------|----------|------------|
| 辞书形 | 読ませる | 食べさせる |
| ます形 | 読ませ**ます** | 食べさせ**ます** |
| て形 | 読ませ**て** | 食べさせ**て** |
| た形 | 読ませ**た** | 食べさせ**た** |
| ない形 | 読ませ**ない** | 食べさせ**ない** |

课文例句就是て形用法：

```
親を心配させ**て**はいけない。
やりすぎて体を壊し、親を心配させ**て**はいけない。
```

## 四、句子结构：谁让谁做什么？

### 4.1 基本框架

```
[使役者] は [被使役者] ＋助词＋ [动作/对象] ＋ 使役动词
```

| 成分 | 说明 | 例 |
|------|------|-----|
| 使役者 | 发出「让…做」的人 | 母、先生、課長 |
| 被使役者 | 被安排去做的人 | 子供、学生 |
| 使役动词 | 未然形 + せる/させる | 書かせる、行かせる |

### 4.2 本课典型句拆解

**例1（他动词 + に + を）**

```
母親は子供に作文を書かせたり、算数の問題を解かせたりする。
│      │   │    │      │           │        │
使役者  は 被使役 に  宾语  書かせる      解かせる
```

- 書く（他动词）→ 被使役者用 **に**
- 作文（动作对象）→ **を**

**例2（自动词 + を）**

```
親を心配させてはいけない。
│  │      │
被使役 を  心配させる（自动词「心配する」的使役）
```

- 心配する在这里作自动词用法（主语自己担心）→ 被使役者用 **を**
- 直译：「（你）不要使父母进入『担心』的状态」

**例3（许可/自谦）**

```
用事があるので、先に失礼させていただきます。
              │    │
              自己  失礼させる（请允许我失礼/先告辞）
```

- 使役者 = 说话人自己
- 被使役者 = 也是自己（请对方允许自己做）
- 这是**商务自谦**固定表达，不是「让别人失礼」

## 五、重点：为什么有时用「に」、有时用「を」？

这是使役态**最容易错**的地方，也是「为什么」的核心。

### 5.1 教材规则（第10课）

> - 动词是**自动词**时 → 被使役者用 **「を」**
> - 动词是**他动词**时 → 被使役者用 **「に」**

### 5.2 为什么要这样分？

要从「使役句本质上在干什么」来理解。

#### 情况A：他动词使役 —— 被使役者是「动作的执行者」

他动词本身结构是：**谁 が 什么 を 做**

```
子供が作文を書く。
```

变成使役后，**子供** 仍是「写」的执行者，所以用表「动作主体/间接对象」的 **に**：

```
母は子供**に** 作文**を** 書かせる。
     执行者↑    宾语↑
```

类比：就像「教える」「与える」一样，**に** 指向「接受动作的人」。

#### 情况B：自动词使役 —— 被使役者是「原主语」，被「拉进句子里当宾语」

自动词结构是：**谁 が 做/状态变化**

```
子供が公園へ行く。
子供が泣く。
親が心配する。
```

使役者要「让这个人去/哭/担心」，等于把原来自动词的主语 **が** 换成 **を**（使役者成为新主语）：

```
先生は子供**を** 公園へ 行かせる。
     原主语变を↑
母は子供**を** 泣かせる。
子供は親**を** 心配させる。
```

**为什么用「を」而不是「に」？**

因为在日语语法里，使役自动词时，被使役者被视为整个使役动作的**直接作用对象**——就像「魚を焼く」里的「魚」一样，是「被施加『去』『哭』『担心』这一状态/行为」的对象。

```
普通句：子供が行く        （孩子是主语）
使役句：子供を行かせる    （孩子变成被作用对象）
```

### 5.3 对照表（必背）

| 动词类型 | 原句 | 使役句 | 被使役者助词 |
|----------|------|--------|--------------|
| 他动词 | 子供**が** 作文**を** 書く | 母は 子供**に** 作文**を** 書かせる | **に** |
| 他动词 | 学生**が** 問題**を** 解く | 先生は 学生**に** 問題**を** 解かせる | **に** |
| 自动词 | 子供**が** 公園へ 行く | 母は 子供**を** 公園へ 行かせる | **を** |
| 自动词 | 子供**が** 泣く | 母は 子供**を** 泣かせる | **を** |
| 自动词 | 親**が** 心配する | 子供は 親**を** 心配させる | **を** |

### 5.4 判断流程（做题用）

```
第一步：看使役动词是自动词还是他动词
        ↓
   有を宾语？→ 多半是他动词 → 人 + に
        ↓
   无を、主语自发？→ 自动词 → 人 + を
```

**快速检验法：** 把使役形还原成基本形，看原来用 **が** 还是 **を**：

```
行かせる → 行く（子供が行く）→ 自动词 → 子供を行かせる ✅
書かせる → 書く（子供が作文を書く）→ 他动词 → 子供に作文を書かせる ✅
```

## 六、和其他语法的区别（避免混淆）

### 6.1 使役 vs 被动

| | 使役 | 被动 |
|---|------|------|
| 形式 | 読**ませる** | 読**まれる** |
| 含义 | 让（人）读 | 被读 |
| 例 | 先生は学生**に**本を読ませる | 学生は先生**に**本を読まれる |

**为什么形式有点像？** 都接在未然形后面，但助动词不同（せる vs れる），语义完全相反。

### 6.2 使役 vs 使役被动（第11课预告）

使役被动 = 被强迫做某事：

```
子供は母**に** 作文**を** 書**かされる**。
（孩子被妈妈逼着写作文）
```

使役者「母」用 **に**，和使役句一样；但动词是 **書かされる**（写 + 使役 + 被动），表示孩子**受害/被迫**。

### 6.3 使役 vs ～てあげる/～てもらう

| 表达 | 侧重点 |
|------|--------|
| 子供に作文を**書かせる** | 使役者主导，「安排/命令去做」 |
| 子供に作文を**書いてあげる** | 说话人替孩子写，孩子受益 |
| 子供**に**作文を**書いてもらう** | 请孩子写，说话人受益 |

## 七、本课例句精讲

### 7.1 课文

| 日语 | 语法点 |
|------|--------|
| 子供に作文を**書かせ**たり、算数の問題を**解かせ**たりする | 他动词 + **に** + を |
| 親を**心配させ**てはいけない | 自动词 + **を** + てはいけない |
| 親を**怒らせ**ることもある | 自动词「怒る」+ **を** |

### 7.2 练习/会话扩展

| 日语 | 说明 |
|------|------|
| 先生は学生**に**会話の練習**を**させます | する → させる，他动词结构 |
| これは父**が**わたし**に**読ませた本です | 他动词「読む」，**に** |
| 彼**に**やらせれば、もっと上手にできます | やる → やらせる |
| いつも子供**を**泣かせるのはよくない | 自动词「泣く」，**を** |
| 弟**を**公園まで**来させ**ます | 自动词「来る」，**を**；まで表目的地 |

## 八、常见错误与纠正

| 错误 | 原因 | 正确 |
|------|------|------|
| 子供**を** 作文を書かせる | 他动词却用了を | 子供**に** 作文を書かせる |
| 子供**に** 公園へ行かせる | 自动词却用了に | 子供**を** 公園へ行かせる |
| 読**める**（能读）当使役 | 与可能态混淆 | 読**ませる**（让读） |
| 食べ**る** → 食べ**る**せる | 一段动词多加错 | 食べ**させる** |
| 来**る** → 来**る**せる | カ变未单独记 | **来させる** |

## 九、自测练习

把下列句子改成使役句（答案见下方折叠思路）：

1. 子供が宿題をする。→ 母は ________。
2. 学生が公園で走る。→ 先生は ________。
3. 太郎が本を読む。→ 父は ________。
4. 子供が泣く。→ 母は ________。

**参考答案：**

1. 母は 子供**に** 宿題**を** **させる**。（する→させる，他动词结构）
2. 先生は 学生**を** 公園で **走らせる**。（走る自动词→を）
3. 父は 太郎**に** 本**を** **読ませる**。（読む他动词→に）
4. 母は 子供**を** **泣かせる**。（泣く自动词→を）
